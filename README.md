# lifestoryai
LifeStory AI (জীবনের গল্প AI) is a modern AI-powered storytelling platform where users can create, read, rewrite, and share life stories in both English and Bangla. এটি একটি আধুনিক AI-চালিত গল্পের প্ল্যাটফর্ম, যেখানে ব্যবহারকারীরা বাংলা ও ইংরেজিতে জীবনের গল্প লিখতে, AI দিয়ে নতুন গল্প তৈরি করতে, সম্পাদনা করতে এবং সবার সঙ্গে শেয়ার করতে পারবেন.
